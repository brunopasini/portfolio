Projetos

DoctorCare

Projeto desenvolvido utilizando JavaScript, CSS e TypeScript onde o usuário pode agendar consultas médicas. Ofertando alguns serviços como, agendamento de consultas médicas, depoimentos dos clientes e contatos. Totalmente responsivo e acessível.

Widget

Neste projeto foi desenvovido um botão Widget utilizando no Front-end JavaScript, CSS, React e React Native e no Back-end Node.js, a funcionalidade utiliza diversas bibliotecas como tirar print, disparar e-mails entre outros, mantendo o padrão de responsabilidade e acessibilidade. O deploy ocorreu na plataforma Vercel e o armazenamento no Railway.
